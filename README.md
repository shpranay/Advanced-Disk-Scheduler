# Advanced Disk Scheduling Simulator

This project is a Python-based simulator for various disk scheduling algorithms. It includes:

- **FCFS (First-Come-First-Serve)**
- **SSTF (Shortest Seek Time First)**
- **SCAN**
- **C-SCAN**
- **LOOK**
- **C-LOOK**
